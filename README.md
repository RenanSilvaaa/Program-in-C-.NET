# Gerenciador  
Esse projeto é um programa básico em C# .NET para salvar senhar e visualizar-las. Além de fazer consultas a itens/produtos, 
checando preços e exportando informações a um bloco de nota

# Funcionalidades
- Exportação de informações
- Visualização de senhas criadas

# Tecnologia
- Visual Studio Code
- Dotnet 8.0.418
